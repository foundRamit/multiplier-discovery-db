# Multiplier Discovery Database
A structured dataset for AI-driven hardware architecture discovery.

## Contents
- **/corpus**: Academic theory and innovation notes.
- **/data/architectures**: Structural metadata for 15 multiplier variants.
- **/data/performance**: Benchmarks (Area, Power, Latency).
- **/rtl_samples**: Verilog source code.

## Quick Start
1. Run `python scripts/validate_schema.py` to check architecture integrity.
2. View `data/performance/master_benchmarks.csv` for the latest synthesis results.
